Privacy Policy

This Privacy Policy applies to the Chrome extension Ephemeral Cypher Companion.

Purpose

Ephemeral Cypher Companion allows users to use the content of the web page they are currently viewing as cryptographic key material for Ephemeral Cypher.

Data Processing

When the user explicitly activates the extension, the extension temporarily accesses the current browser tab and processes the page content locally in the user's browser.

The page content is normalized and converted into a cryptographic fingerprint for use with Ephemeral Cypher.

Data Collection

Ephemeral Cypher Companion does not collect, store, sell, or share personal information.

The extension does not send page content, browsing history, URLs, or generated cryptographic key material to the developer or to third-party servers.

Data Transmission

All page-content processing is performed locally in the user's browser. No third-party relay service is used for processing the selected web page.

Permissions

The extension uses the activeTab permission to temporarily access only the browser tab that the user explicitly activates.

The scripting permission is used to process the selected page within that tab for the sole purpose of generating cryptographic key material for Ephemeral Cypher.

Third Parties

User data is not sold, transferred, or used for advertising, analytics, profiling, or any purpose unrelated to the extension's single purpose.

Changes to This Policy

This Privacy Policy may be updated if the functionality of Ephemeral Cypher Companion changes.

Contact

For questions regarding this Privacy Policy, please contact the developer through the Ephemeral Cypher GitHub repository.
